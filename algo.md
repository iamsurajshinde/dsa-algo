## ✅ 1. Searching Algorithms
Used when you need to find something.
- Linear Search
- Binary Search
- Ternary Search
- Exponential Search
- Interpolation Search
- Jump Search

## ✅ 2. Sorting Algorithms
Used when order matters.
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort
- Shell Sort
- Tim Sort

## ✅ 3. Two Pointer & Sliding Window
Used mainly on arrays / strings.
- Two Pointer Technique
- Sliding Window (fixed window, variable window)

## ✅ 4. Prefix / Cumulative Techniques
Used for range queries and subarray problems.
- Prefix Sum
- Difference Array
- Running Sum

## ✅ 5. Hashing Based Algorithms
Used when fast lookup is needed.
- Hash Map / Hash Set based approach
- Frequency counting
- Complement lookup (like two-sum pattern)

## ✅ 6. Greedy Algorithms
Used when you make the best local decision.
- Activity Selection
- Interval Scheduling
- Job Sequencing
- Huffman Coding
- Minimum Platforms
- Fractional Knapsack

## ✅ 7. Divide and Conquer
Break problem into smaller parts.
- Merge Sort
- Quick Sort
- Binary Search
- Closest Pair of Points
- Strassen's Matrix Multiplication

## ✅ 8. Dynamic Programming (DP)
Used when overlapping sub-problems exist.

### Classical DP
- Fibonacci DP
- Knapsack (0/1, unbounded)
- Coin Change
- LIS / LCS
- Matrix Chain Multiplication
- Edit Distance
- Palindrome DP

### DP Patterns
- 1D DP
- 2D DP
- State machine DP
- Subset DP
- Bitmask DP
- Digit DP

## ✅ 9. Backtracking Algorithms
Used when you must try all valid possibilities.
- N-Queens
- Sudoku Solver
- Permutations
- Combinations
- Subsets
- Word Search

## ✅ 10. Graph Algorithms

### Traversal
- BFS
- DFS

### Shortest Path
- Dijkstra
- Bellman-Ford
- Floyd-Warshall
- 0-1 BFS

### Minimum Spanning Tree
- Kruskal
- Prim

### Connectivity / Components
- Union-Find (Disjoint Set)

### Topological Sort
- Kahn's Algorithm
- DFS based Topological Sort

### Advanced
- Bridges & Articulation Points
- Strongly Connected Components (Kosaraju, Tarjan)
- Bipartite Check
- Maximum Flow (Ford-Fulkerson, Edmonds-Karp)

## ✅ 11. Tree Algorithms
- Tree Traversals (inorder, preorder, postorder, level order)

- Balanced Tree checks

- Trie algorithms
